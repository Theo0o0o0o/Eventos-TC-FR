# Eventos TC

Versão com redesign glassmorphism e integração com Supabase.

## Stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- React Router
- TanStack Query
- React Hook Form + Zod
- Supabase (Auth + Database)

## Rodar localmente

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```

## Variáveis de ambiente

Crie um arquivo `.env.local` ou ajuste o `.env` com:

```bash
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```
