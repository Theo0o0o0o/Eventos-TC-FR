# Redesign editorial / poster notes

## Layout geral
- Estrutura original mantida, com redesign visual em linguagem editorial / pôster.
- Hero redesenhado como painel principal com grid visível, headline de grande impacto, imagens em moldura estilo papel e composição assimétrica.
- Header convertido para navegação em pills, com bloco de marca mais forte e aparência premium.
- Cards e painéis transformados em superfícies tipo papel, com profundidade, sombra leve e bordas sutis.
- Rodapé refeito em base azul escura com grid suave, mantendo os conteúdos principais.

## Direção de tipografia
- Base com fonte sans-serif para leitura corrida.
- Headings com fonte display bold e uppercase para mais presença visual.
- Hierarquia reforçada com títulos grandes, labels pequenas em uppercase e corpo de texto mais limpo.

## Paleta sugerida
- Azul vibrante como cor principal.
- Creme / off-white como base do sistema.
- Grafite para títulos e textos principais.
- Amarelo quente como cor de destaque pontual para botões e elementos de ênfase.

## Estilo dos componentes
- Botões arredondados em pills, com contraste forte e resposta rápida.
- Inputs, selects e textareas com acabamento clean tipo papel.
- Cards com grid sutil e sensação física leve.
- Popovers, dropdowns e modais alinhados com a mesma linguagem visual.
- Imagens apresentadas em molduras tipo polaroid / papel, com pequenas rotações controladas.

## Espaçamento, grid e responsividade
- Mais respiro vertical nas secções principais.
- Grid de fundo visível, mas com opacidade controlada para não prejudicar legibilidade.
- Elementos decorativos reduzidos no mobile para manter clareza.
- Animações encurtadas e mais leves para melhorar fluidez.
